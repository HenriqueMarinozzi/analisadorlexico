#Visão Geral

Este projeto desenvolve um analisador léxico voltado para expressões aritméticas em notação polonesa reversa (RPN), incluindo também o processamento dos tokens e a geração de código Assembly compatível com o simulador Cpulator ARMv7 DEC1-SOC (v16.1).
O sistema lê um arquivo de texto contendo uma expressão por linha, executa a análise léxica, interpreta os tokens obtidos, apresenta o resultado do processamento e gera o código Assembly referente à última execução realizada.
